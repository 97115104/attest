sdk (recommended)

The attest-client npm package provides deterministic attestation creation. It eliminates URL hallucination by returning exact server-generated URLs as structured objects. Zero dependencies.

install

npm install attest-client

one-shot

import { attest } from 'attest-client';

const result = await attest({
  content_name: 'README.md',
  model: 'claude-opus-4',
  role: 'collaborated',
  author: 'my-agent'
});
// result.urls.short → exact short URL from server
// result.urls.verify → exact verify URL from server

client instance (reusable)

import { createClient } from 'attest-client';

const client = createClient({ author: 'my-agent' });
const r1 = await client.create({ content_name: 'file1.md', model: 'claude-opus-4' });
const r2 = await client.create({ content_name: 'file2.md', model: 'claude-opus-4' });
const metrics = await client.metrics();
const spec = await client.discover();

cli

npx attest --content README.md --model claude-opus-4 --role collaborated
npx attest --metrics
npx attest --discover
npx attest --content file.md --json  # raw JSON output

Identifies itself via User-Agent: attest-client/2.0.0 for tracking. npmjs.com/package/attest-client

executable examples

create attestation (minimal)

GET https://attest.97115104.com/api/create?content_name=README.md

create attestation (full)

GET https://attest.97115104.com/api/create?content_name=README.md&model=claude-opus-4&role=collaborated&author=my-agent&content=hello+world

upload file

curl -F "file=@document.md" \
  -F "role=collaborated" \
  -F "model=claude-opus-4" \
  -F "author=my-agent" \
  https://attest.97115104.com/api/create-upload

attest a url

curl -X POST https://attest.97115104.com/api/create-url \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com","role":"generated","model":"gpt-4","author":"my-agent"}'

shorten a verify url

curl -X POST https://attest.97115104.com/api/shorten \
  -H "Content-Type: application/json" \
  -d '{"data":"eyJ2ZXJzaW9uIjoiMi4wIn0="}'

decode attestation (client-side)

JSON.parse(atob(data))

get platform metrics

GET https://attest.97115104.com/api/metrics
recommended agent workflow
1. Install: npm install attest-client (or use raw HTTP)
2. Create content (code, docs, articles, etc.)
3. Call attest() or GET /api/create with content_name, model, role, author
   - SDK returns exact URLs, no hallucination risk
   - Optionally pass content= for SHA-256 hash verification
4. Receive attestation JSON + verify URL + short URL
5. Store attestation alongside content:
   - HTML: <meta name="attestation" content="{base64}">
   - Frontmatter: attestation: {short_url}
   - Sidecar: /attestations/{filename}.json
6. Anyone can verify at the verify URL
