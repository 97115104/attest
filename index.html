attest

Transparent AI attribution for the modern web

▸ What is this?

attest is an open-source protocol that lets you declare whether content was written by a human, co-authored with AI, or fully AI-generated. Add a single frontmatter line to your post and the system produces a verifiable, signed attestation.

▸ One-line usage

Add an attribution field to any markdown frontmatter:

---
title: My Blog Post
attribution: collab claude opus 4
---

Types: human · collab <model> · ai <model>

▸ Programmatic (one line)

curl "https://attest.97115104.com/api/create?content_name=My+Post&model=claude-opus-4&role=collaborated"

Returns JSON with the attestation + a verify URL. That's it.

▸ How it works

Pipeline
  1. You write content and declare attribution: collab claude opus 4
  2. A script hashes the content (SHA-256) and signs it with HMAC
  3. The attestation JSON is stored alongside your content
  4. Anyone can verify via the data URL — no server needed
