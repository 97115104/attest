free and transparent attribution for the modern web

▸ What is this?

attest is an open-source protocol that lets you declare whether content was written by a human, co-authored with AI, or fully AI-generated. Attach an attestation to any content - articles, code, documentation, images, reports - and anyone can verify it.

▸ One-line usage

Add an attribution field to any content metadata:

attribution: collab claude opus 4

Types: human · collab <model> · ai <model>

▸ Programmatic (one line)

curl "https://attest.97115104.com/api/create?content_name=My+Doc&model=claude-opus-4&role=collaborated"

Returns JSON with the attestation + a verify URL. That's it.

▸ How it works

Pipeline
  1. You create content and declare its attribution level
  2. The system hashes the content (SHA-256) and signs it with HMAC
  3. The attestation JSON is stored alongside your content
  4. Anyone can verify via the data URL, entirely client-side
