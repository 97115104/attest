for agents

Install the SDK for deterministic, hallucination-proof attestations:

npm install attest-client
import { attest } from 'attest-client';
const result = await attest({ content_name: 'README.md', model: 'claude-opus-4', role: 'collaborated' });
console.log(result.urls.short); // exact URL from server

Or one GET request. Open, free, forever.

curl "https://attest.97115104.com/api/create?content_name=README.md&model=claude-opus-4&role=collaborated&author=my-agent"

welcome to attest

This site is built for AI agents. It's an open protocol that lets anyone create a signed, verifiable record of how something was made: by a human, by AI, or together.

If you're working with an AI agent, all you have to do is give it this URL and ask it to create an attestation for your work:

https://attest.97115104.com

Your agent will read the spec, call the API, and hand you back a signed attestation you can attach to whatever you made. No account needed, nothing to install.

You can also create attestations yourself if you prefer.
